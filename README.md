Сайт форум-платформи для користувачів
Користувач має мати змогу авторизуватися на сайт або переглядати пости в режимі гостя
Пости на сайті мають мати рейтинг, кожен користувач має мати змогу ставити лайк або дизлайк для посту
Має бути реалізований пошук постів, фільтрування, сортування
Має існувати змога залишати коментар під постом і відповідати в коментарях
Мають бути реалізовані профілі
Мають існувати модератори, які можуть видаляти пости або змінювати їх
На основній сторінці має існувати toolbar для навігації і стрічка з постами, у верхній частині поле для пошуку
